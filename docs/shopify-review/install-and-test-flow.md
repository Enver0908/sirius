# Sirius - Install and Test Flow

1. Development store uzerinden Sirius uygulamasini acin.
2. Shopify izin ekranini onaylayin.
3. Ilk kurulumda setup akisi acilir.
4. `Sirius Pro` planini secin.
5. Claude, GPT veya Gemini icin bir API anahtari baglayin.
6. Dashboard acildiginda yeni chat baslatin.
7. Sol panelden gecmis konusmalari test edin.
8. Model secimini degistirip key yok ve kota bitti senaryolarini dogrulayin.
